void inserta_obstaculo ( Escenario e,
int tipoObstaculo,
int x,
int y,
int w,
int h )

Añade un obstáculo definido mediante el rectángulo x,y,w,h.

Precondición
Para que funcione correctamente es necesario que las coordenadas y tamaños del obstáculo den múltiplos del tamaño de bloque.
Nota
Los obstáculos se pueden ir superponiendo hasta terminar de modelar las colisiones en el mapa como se quiera. Tiene mayor orden de importancia siempre el último obstáculo añadido.
Parámetros
eEscenario al cual se le va a añadir un obstáculo.
tipoObstaculo"0" Si no es un obstáculo. "1" Si no se puede traspasar por nadie. "2" Si solo una parte de los entes puede traspasarlo.
xCoordenada horizontal superior izquierda.
yCoordenada vertical superior izquierda.
wAnchura en pixeles del obstáculo.
hAltura en pixeles del obstáculo.
